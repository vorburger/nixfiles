# Nix Wiki

For historical reasons, there is https://wiki.nixos.org and https://nixos.wiki.

The one on `nixos.org` is the "[current](https://wiki.nixos.org/w/index.php?hidebots=1&translations=filter&limit=50&days=7&enhanced=1&title=Special:RecentChanges&reviewStatus=unpatrolled&urlversion=2)" one to use.

The one on `nixos.wiki` [still gets some updates, but mostly spam](https://nixos.wiki/wiki/Special:RecentChanges?hidebots=1&limit=50&days=30&enhanced=1&urlversion=2).

They both use MediaWiki.
